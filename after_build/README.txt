Purpose of this directory:
	These are enhancements that are needed after the installation of the virtual machine. 
	

- configs
	This directory has all of the credentials and or setups needed for various parts of the system

- pma
	This is the PHPMyadmin install for the VM. Following the instructions from the README in the root folder, you will drage
	and drop this inside of the webroot of the vm to use this program.

- ppk_Key
	This is the ssh key to connect with the vm; either through PuTTY or FileZilla. Not needed to use FileZilla, but neccessary for PuTTY.
	
- test
	This is the test directory to drop into the webroot of the VM to make sure that the installation went as expected.
